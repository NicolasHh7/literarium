# Literarium

Este projeto consiste no desenvolvimento de uma plataforma web onde usuários podem criar seus próprios livros de forma prática e criativa. O sistema utiliza Inteligência Artificial para auxiliar na correção de textos e na geração de imagens, como capas personalizadas, tornando o processo de escrita mais acessível, interativo e profissional.

A proposta é oferecer um ambiente completo para que autores iniciantes ou experientes possam escrever, revisar, ilustrar e exportar suas obras em um só lugar.

## Funcionalidades do Site

- **Criação de livros**  
  O usuário pode escrever e organizar seu próprio livro diretamente na plataforma.

- **Correção de texto com IA**  
  A Inteligência Artificial ajuda a revisar ortografia, gramática e melhorar a escrita.

- **Geração de imagens com IA**  
  Possibilidade de criar imagens automaticamente, como capas para os livros.

- **Múltiplos projetos**  
  O usuário pode criar e gerenciar mais de um livro ao mesmo tempo.

- **Download do livro**  
  Permite baixar o livro finalizado em formato digital.

- **Sistema de login e cadastro**  
  Cada usuário possui uma conta para salvar e acessar seus projetos com segurança.
